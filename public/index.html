③ 시트립 브라우저 열기 & 시작

아래 ▶ 시트립 브라우저 열기를 누르면 자동화용 창이 뜹니다. 그 창에서 로그인 → Account 그룹선택(1210/1311) → Mapping center → Room Mapping 화면을 띄운 뒤, 아래 ③-2 자동 스캔으로 진행하세요.

수동/디버그:

③-2 자동 스캔 (codes.txt → 룸별 추천 자동 수집)

스캔 전 준비(중요): ③ 브라우저 열기 → 시트립 로그인 → Account에서 그룹 선택(1210/1311) → Mapping center → Room Mapping 화면을 띄워두세요. 그 화면에서 호텔코드만 바꿔가며 자동 조회→룸 모달→AMS 추천을 모읍니다. 최종 [Mapping] 클릭은 사람.

쿨다운 중인(최근에 결과 없던/매핑한) 코드는 자동으로 건너뜁니다.

행을 클릭하면 우리 룸 ↔ 추천 후보를 모든 컬럼으로 비교합니다.

④ 알고리즘 설정

점수 가중치·임계값. 바꾸면 다음 추천부터 반영됩니다.

결과 없음·매핑한 호텔코드는 이 기간 동안 자동 스캔에서 건너뜁니다. 0=항상 재검색.
