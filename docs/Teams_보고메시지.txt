📢 [공유] AMS(AI 매핑 시스템) 프로토타입 개발 보고

안녕하세요, 콘텐츠·매핑팀입니다.
OTA 채널 간 객실 매핑을 자동화하는 AMS 프로토타입을 개발하여 현재 정상 작동 중입니다.

✅ 주요 내용
• 시트립 호텔코드 → 미매핑 룸 자동 조회 → AMS 점수화 → 추천까지 자동화
• AI 외부검증을 통해 매핑 신뢰도를 올리는 작업을 가능하게 했습니다
• 최종 매핑 확정은 사람이 통제(오매핑 방지)

⚠️ 비용 참고
• AI 외부검증은 Claude API Key를 사용하므로, Claude 구독료 외 별도의 비용이 청구될 수 있습니다
• (절감 장치 적용: 80% 이상만 검증, 묶음 검증, 호출 제한 — AI를 끄면 비용 0으로도 운용 가능)

🔄 인계 및 향후 계획
• 프로토타입 코드를 명세서화하여 콘텐츠팀 Karl에게 넘겼습니다
• Karl이 매핑 날리지를 통해 고도화할 예정입니다
• 빠른 시일 내 다시 보고드리겠습니다

📢 AMS(AI 매핑 시스템) 프로토타입 개발 보고

OTA 객실 매핑을 자동화하는 AMS 프로토타입을 개발해 현재 작동 중입니다.
AI 외부검증을 통해 매핑 신뢰도를 올리는 작업도 가능하게 했습니다.

⚠️ AI 외부검증은 Claude API Key를 사용해 Claude 구독료 외 별도 비용이 청구될 수 있습니다.

프로토타입 코드는 명세서화하여 콘텐츠팀 Karl에게 인계했으며, Karl이 매핑 날리지를 통해 고도화할 예정입니다. 빠른 시일 내 다시 보고드리겠습니다.
